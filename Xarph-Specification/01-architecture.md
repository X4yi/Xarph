# Arquitectura General

## Componentes principales

### xarph-wm

Fork de Niri.

Responsabilidades:

- Wayland.
- Gestión de ventanas.
- Workspaces.
- Animaciones.
- Protocolos internos.

### xarph-shell

Fork de QuickShell.

Responsabilidades:

- Barra.
- Widgets.
- Fondo.
- Dashboard.
- Notificaciones.
- Escritorio.

### xarph-settings

Centro de configuración.

### xarph-sdk

SDK oficial.

### xarph-launcher

Lanzador oficial.

### xarph-lock

Pantalla de bloqueo.

### Process Admin 

Frontend gráfico para btop y herramientas del sistema.
