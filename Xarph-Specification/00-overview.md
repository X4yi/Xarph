# Xarph Desktop Environment

## Resumen

Xarph es un entorno de escritorio modular para Linux construido sobre Wayland.

Objetivos:

- Ligero.
- Potente.
- Modular.
- Integrado profundamente con Linux.
- Compatible con gaming y productividad.
- Extensible mediante SDK.
- Capaz de funcionar completamente sin IA.

## Filosofía

Xarph no busca imponer un flujo de trabajo.

El usuario decide cómo utilizarlo.

La personalización es una consecuencia de la arquitectura, no el objetivo principal.
